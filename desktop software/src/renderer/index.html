How to get a Gemini API key
  1. Visit aistudio.google.com and sign in with your Google account.
  2. Click Get API key in the left sidebar.
  3. Create a new key or copy an existing one.
  4. Paste it in the field above and click Save.
  5. The status pill in the top bar will turn green.
